# Attendance Application

An attendance app that will save a list of students by year/cohort (fed into app using text file or copy/paste) and assign them to classes. In a class I can simply check a box to indicate if they are present or not and have that data stored to a db (sqlite should be fine enough).
Ideally the app would be frictionless to use. Check a box (with the ability to uncheck if a student walks in late).
